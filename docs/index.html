Core Features

First-version content, with icon treatment aligned to the second and third versions.

Privacy First

Runs entirely on your machine. Your code never leaves your local environment, ensuring maximum security and compliance.

Rich Aesthetics

Designed for developers who care about visual excellence. A premium CLI experience with syntax highlighting and clear diffs.

Intelligent Context

Advanced XML-based codebase analysis provides the AI with perfect context, resulting in highly accurate code edits.

Model Profiles

Seamlessly switch between different models like Qwen, DeepSeek, and Codestral with dedicated configuration profiles.

Safety Drills

Always see a full preview of changes before they are applied. Built-in git integration ensures you can rollback anytime.

Native Speed

Optimized for local execution using llama.cpp. Low latency interactions that keep you in the flow.
